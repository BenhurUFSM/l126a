### Aula 9

Vimos [if com else](if.md) e [funções que retornam valores](funcao.md).

#### Exercícios

1. Faça uma função que retorna o maior entre dois valores recebidos.
2. Faça uma função que retorna o maior entre três valores recebidos. Use a função anterior.
1. Faça uma função que retorna o valor intermediário entre três valores recebidos.
2. Faça uma função que retorna o quadrado do número recebido.
1. Faça uma função que retorna o n-ésimo número da sequência de Fibonacci, recebendo n como parâmetro. Use a função `main` abaixo para testar:
```c
   int main()
   {
       puts("+----+------+");
       puts("|  n | fibo |");
       for (int i = 0; i < 10; i++) {
           printf("| %2d | %4d |\n", i+1, fibo(i+1));
       }
       puts("+----+------+");
   }
```
1. Faça uma função que recebe um número e retorna maior índice na sequência de Fibonacci que corresponde a um número que não é maior que o número fornecido. Use a função do exercício anterior. Por exemplo, se receber o valor 10 deve retornar 6, porque o 6º número da sequência de Fibonacci é 8 (não maior que 10) e o 7º é 13 (maior que 10).
1. Faça uma função que recebe em número e retorna verdadeiro (1) se o número for interessante, e 0 se não for. Implemente duas versões, uma para cada definição de número interessante da lista de exercícios anterior. Refaça aqueles exercícios usando essas funções.
1. Faça uma função que calcula a raiz quadrada do número recebido como argumento. A raiz retornada deve ser o maior número cujo quadrado não é maior que o número dado. Por exemplo, a raiz de 15 nessa aproximação é 3.
